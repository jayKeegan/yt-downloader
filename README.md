# YouTube Downloader
Download videos from YouTube using TypeScript.

All videos will be stored in the videos folder.

**Setup Guide**
- `git clone git@github.com:jayKeegan/yt-downloader.git` - clone repository
- `npm install` - insall required dependences
- `npm install ts-node -g` - install typescript globally to run the program
- open index.ts and fill in url & extension variables with their respective values
- `ts-node index.ts` - start the program
